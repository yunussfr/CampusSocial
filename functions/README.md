# CampusConnect Cloud Functions

This folder contains the first backend trigger set for Phase 6.

Implemented triggers:

1. `onEventJoin`
2. `onEventLeave`
3. `onCommunityJoin`
4. `onCommunityLeave`
5. `onMessageSent`
6. `onListingSave`
7. `onUserFollow`
8. `onUserUnfollow`

Counters are owned by Cloud Functions. Client code must not update counter fields directly.
